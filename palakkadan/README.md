This is a premium, high-performance single-page sales engine for **Palakkadan.in**, built by DevBridge Kerala. Designed for maximum conversion on WhatsApp and mobile.

## Project Summary
- **Business**: Palakkadan
- **Build Date**: April 2, 2026
- **System**: Master Build Engine v14.1
- **Focus**: Heritage, Organic Living, Authentic Palakkad Taste.

## File Structure
- `index.html`: Main landing page with critical CSS and DVK Pixel integrated.
- `/images/`: High-resolution product assets (Hero, Services, Gallery).
- `/seo/`: Full infrastructure (robots.txt, sitemap.xml, .htaccess) for search and security.

## Local Preview
1. Open `index.html` in any modern browser.
2. Note: Images and styles are optimized for a natural, premium feel.

## Go-Live Instructions
1. Upload all files (including folders) to your domain root (public_html).
2. Copy contents of the `/seo/` folder to the domain root:
   - Move `robots.txt` and `sitemap.xml` to the root.
   - Move `.htaccess` to the root (Apache servers).
3. Update the `sitemap.xml` with your final domain name.
4. Replace the placeholder testimonials in `index.html` with real client feedback before final launch.
